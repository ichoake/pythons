## üö¶ Quick Commands

### List all Python files
```bash
ls -la ~/Documents/python/*.py | wc -l
```

### Find scripts that use specific library
```bash
grep -l "import openai" ~/Documents/python/*.py
grep -l "import whisper" ~/Documents/python/*.py
grep -l "from PIL" ~/Documents/python/*.py
```

### Find all classes defined
```bash
grep "^class " ~/Documents/python/*.py
```

### Find all functions defined
```bash
grep "^def " ~/Documents/python/*.py
```

### Run a specific script
```bash
python ~/Documents/python/script-name.py
```

### View script dependencies
```bash
head -20 ~/Documents/python/script-name.py | grep "^import\|^from"
```

## üìã Checklist for Common Tasks

### Before Processing a Large Batch
- [ ] Backup original files (15day.py, backupcsv.py)
- [ ] Run analysis script (analyze-*.py)
- [ ] Create dry-run version (AutoOrganizeDryRun)
- [ ] Review changes before applying

### Before Uploading Content
- [ ] Verify file format (appropriate for platform)
- [ ] Check metadata (analyze-metadata.py)
- [ ] Add text/captions if needed (add_text.py)
- [ ] Create thumbnail if needed (custom-thumbnail.py)

### Before Archiving Files
- [ ] Deduplicate (intelligent_dedup.py)
- [ ] Consolidate versions (consolidate_versions.py)
- [ ] Create index (downloads-index-builder.py)
- [ ] Backup metadata (backupcsv.py)
