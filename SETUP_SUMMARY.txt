INSTALLATION STEPS
================================================================================

1. Run the setup script:
   chmod +x macos-ai-setup.sh
   ./macos-ai-setup.sh

2. Configure API keys:
   cp ~/.env.d/ai-services.env.template ~/.env.d/ai-services.env
   nano ~/.env.d/ai-services.env

3. Restart terminal:
   source ~/.zshrc

4. Verify installation:
   ai-load-env
   python verify-ai-setup.py

QUICK START EXAMPLE
================================================================================

After setup, create a file test-ai.py:

```python
from openai import OpenAI
from anthropic import Anthropic
import google.generativeai as genai
import os

# OpenAI
openai_client = OpenAI()
gpt_response = openai_client.chat.completions.create(
    model="gpt-4-turbo-preview",
    messages=[{"role": "user", "content": "Hello from GPT!"}]
)
print("GPT-4:", gpt_response.choices[0].message.content)

# Claude
claude_client = Anthropic()
claude_response = claude_client.messages.create(
    model="claude-3-opus-20240229",
    max_tokens=100,
    messages=[{"role": "user", "content": "Hello from Claude!"}]
)
print("Claude:", claude_response.content[0].text)

# Gemini
genai.configure(api_key=os.environ['GOOGLE_API_KEY'])
gemini = genai.GenerativeModel('gemini-pro')
gemini_response = gemini.generate_content("Hello from Gemini!")
print("Gemini:", gemini_response.text)
```

Run with:
  ai-load-env && python test-ai.py
